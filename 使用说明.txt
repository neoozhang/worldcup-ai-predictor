  📦 第一步：安装 Node.js（必需）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1. 打开 https://nodejs.org
  2. 下载左边那个 LTS 版本（推荐 18.x 或更高）
  3. 双击安装包，一路点"下一步"即可
  4. 验证安装：打开"命令提示符"，输入 node -v ，显示版本号就OK


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  🚀 第二步：启动程序
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  方法一（推荐）：双击 start.bat → 自动安装依赖 + 启动

  方法二（命令行）：
    1. 在此文件夹空白处 Shift+右键 → "在此处打开 PowerShell"
    2. 输入 npm install  →  安装依赖（只需一次）
    3. 输入 npm start    →  启动服务器

  浏览器会自动打开 http://localhost:3000
  没自动打开的话手动输一下这个地址


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  🎮 第三步：开始使用（不配 Key 也能用！）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  即使不配任何 API Key，你也可以：

  ✅ 浏览全部 30+ 场比赛赛程
  ✅ 查看小组积分榜（实时计算）
  ✅ 录入比赛结果（点 ✏️ 修改比分）
  ✅ 使用亮色/暗黑主题切换

  想要 AI 预测功能？继续往下看 ↓


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  🤖 第四步：配置 AI 预测（可选）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  编辑此文件夹里的 .env 文件（用记事本打开即可）
  填入你想用的 AI 的 API Key，配几个用几个！

  ════════════════════════════════════════
  🧠 Claude (Anthropic)
  ════════════════════════════════════════
  在 .env 里写：ANTHROPIC_API_KEY=sk-ant-你的key

  如何获取：
  1. 打开 https://console.anthropic.com
  2. 注册/登录（支持 Google 账号）
  3. 左侧菜单 → API Keys → Create Key
  4. 复制 key，粘贴到 .env 文件

  费用：按量计费，新用户有免费额度

  ════════════════════════════════════════
  🤖 GPT (OpenAI)
  ════════════════════════════════════════
  在 .env 里写：OPENAI_API_KEY=sk-你的key

  如何获取：
  1. 打开 https://platform.openai.com/api-keys
  2. 注册/登录
  3. Create new secret key → 复制 → 粘贴

  费用：按量计费，新用户有 $5 免费额度

  ════════════════════════════════════════
  💎 Gemini (Google)
  ════════════════════════════════════════
  在 .env 里写：GEMINI_API_KEY=AIza-你的key

  如何获取：
  1. 打开 https://aistudio.google.com/apikey
  2. 登录 Google 账号
  3. Create API Key → 复制 → 粘贴

  费用：有免费额度，够日常使用

  ════════════════════════════════════════
  🔍 DeepSeek
  ════════════════════════════════════════
  在 .env 里写：DEEPSEEK_API_KEY=sk-你的key

  如何获取：
  1. 打开 https://platform.deepseek.com
  2. 注册/登录
  3. API Keys → 创建 → 复制 → 粘贴

  费用：非常便宜，充 10 块用很久

  ════════════════════════════════════════
  🫘 豆包 (ByteDance 火山引擎)
  ════════════════════════════════════════
  在 .env 里写：DOUBAO_API_KEY=你的key

  如何获取：
  1. 打开 https://console.volcengine.com/ark
  2. 注册/登录火山引擎
  3. 开通模型服务 → API Key 管理 → 创建 → 复制 → 粘贴

  费用：有免费额度

  ════════════════════════════════════════

  配置完成后重启服务器（关掉重开 start.bat）
  页面上对应的 AI 按钮会自动亮起！

  🔧 常见问题
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Q: 双击 start.bat 闪退？
  A: 可能没装 Node.js，先去 nodejs.org 下载安装

  Q: 浏览器显示"无法连接"？
  A: 确认 start.bat 窗口还开着，端口是 3000

  Q: 点了预测但报错？
  A: 检查 .env 文件里的 Key 是否正确粘贴，不要有多余空格

  Q: 只想用一个 AI？
  A: 完全没问题，.env 里只填一个 Key 就行，其他留空

  Q: 怎么切换 AI 模型？
  A: .env 里加一行，比如 GPT_MODEL=gpt-4.1
     详见 .env.example 文件

  Q: 端口被占用了？
  A: .env 里改 PORT=3001 或其他端口
